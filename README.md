## 기대사항
제가 경험한 Front 프로젝트의 경험은 혼자 였었습니다. 혼자서 학습하고 혼자서 어떤 방향이 좋을까에 대해서 고민하는 부분들이 많았습니다. 처음에는 혼자 하기 때문에 많은 부분들이 습득하는 부분에 있어서 도움이 많이 되었습니다. 

그런데, 이제 다 완성하다 보니까 잘 구축을 한 부분인지, 어떤 부분이 부족한지를 평가받고 조언받고 싶었습니다. 그래서 저는 Front end 채용시장에 관심을 갖게 되었습니다. 

저는 2가지의 기대사항이 있습니다. 

**첫번째**는 좋은 시니어 분을 만나는 것이고, **두번째**는 커뮤니케이션적으로 한번 더 성장하는 부분이 저의 기대사항입니다. 

"좋은 시니어 분을 만나는 것"

*개발적으로 성장할 수 있는 환경과 기술적인 부분에 대해서 배울 수 있는 시니어 분을 만나는 것.*

"커뮤니케이션 성장"

*커뮤니케이션은 일하는데 있어서 가장 중요한 요소라고 생각합니다. 정답도 없다고 생각합니다. 하지만 제가 되고자 하는 부분은 어떤 누구와 일하더라도 성과를 만들어 낼 수 있고, 동료가 되어 줄 수 있는 그런 사람이 되고자 합니다*
